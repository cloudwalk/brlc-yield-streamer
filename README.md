This repository contains [BRLC](https://infinitepay.io/brlc) token yield streaming smart contracts.</br>
[BRLC](https://infinitepay.io/brlc) is a stablecoin created and issued by [Infinitepay](https://infinitepay.io).

## Build and test

``` sh
# Install all dependencies
npm install

# Compile all contracts
npx hardhat compile

# Run all tests
npx hardhat test
```
